# Application "Voting"

This is web-application, that able to create different votes.


## Getting Started

```

1. Navigate to the folder with project, the next write:

2. mvn clean package

3. mvn spring-boot:run
```




### Prerequisites


```
spring boot
java 8
maven 3
```
